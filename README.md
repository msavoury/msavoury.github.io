# msavoury.github.io

Personal website and portfolio for Marcos Savoury - Senior Software Engineer at Block.

## About

This site showcases my professional experience, technical skills, and background as a software engineer specializing in distributed systems and backend development.

## Site Structure

- **Home**: Introduction and bio
- **About**: Professional summary and core competencies
- **Experience**: Detailed work history and education

## Technology

Built with [Jekyll](https://jekyllrb.com/) using the [Indigo](https://github.com/sergiokopplin/indigo) minimalist theme.

## Local Development

To run the site locally:

```bash
bundle install
bundle exec jekyll serve
```

Then open `http://localhost:4000` in your browser.
